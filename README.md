# TSLint Config Flagwind

[![NPM version](https://img.shields.io/npm/v/tslint-config-flagwind.svg?style=flat)](https://www.npmjs.com/package/tslint-config-flagwind)

> A [TSLint config](https://palantir.github.io/tslint/usage/tslint-json/) for [Flagwind Code Guide](https://github.com/flagwind/code-guide)

## Installation

```sh
npm install tslint-config-flagwind --save-dev
```

## Usage

In `tslint.json`:

```json
{
  "extends": "tslint-config-flagwind"
}
```
